HTML

  • HTML is used in coding to create the structure of a webpage using elements or element types. Some common elements include Text, Image, Header, List.
  • The HEAD element contains information, or metadata, that helps the browser to render the page correctly. Elements included within the HEAD are not visible when a user looks at a webpage.
  • The BODY element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.

Git

  • Git is a an open-source version control software
  • git clone "Repository SSH": Copies the selected GitHub repository anywhere on your local directory
  • git status: Checks to see if the current branch is operating smoothly
  • git checkout "Branch name": Changes to the specified branch
  • git add -b "Branch name": Creates a branch with a chosen name
  • git add -A: Adds any changes made to the current branch by firstly staging any changes made to the code
  • git commit -m "Comment": Compiles all changes within the staging area and overwrites all data on the current branch.
  • git pull origin main: Checks to see if your code is in sync with the latest version of a repository
  • git push origin "Branch": Creates a Pull Request (PR) which will request a decision from the project team to merge any updated code to the main repository
